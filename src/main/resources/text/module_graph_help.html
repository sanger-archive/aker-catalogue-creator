Module graph


The To add combo shows all modules listed in the catalogue.

To add a module to the graph, select it from the combo, and then double-click in the graph at the position you want to add the module.

The same module cannot be added to the graph twice.

Modules must be positioned below the START node and above the END node.


To add a path, right-click on a module and drag to another module.

The end-node of a path must be positioned lower in the graph than the start-node.

Shift-click on a path to toggle it as a default path. Default paths are shown in green.


Click on a module or a path to select it.

Press DELETE to delete a selected path or module.

Drag a module to reposition it. Modules must remain between the START and END nodes. The start-node of paths must stay above the end-node.


The Validate option in the Edit menu will check your graphs are laid out correctly.


The default layout for a module graph is generated based on the paths between modules when you load a catalogue file. Use Auto layout to recalculate the default layout for your current graph.


Use Resize graph window to adjust the graph bounds to accommodate the current positions of the modules.
